# UI Guidelines Reference

## Product Feel

Keep the interface:

- soft
- calm
- desktop-like
- Chinese-first
- glassmorphism-inspired

## Preferred Direction

Use the scenic `v4` Chinese direction as the current reference:

- layered background atmosphere
- dark outer shell for depth
- bright glass panels for readability
- calm blue highlights
- rounded cards and gentle contrast

## Avoid

- admin dashboard look
- harsh neon accents
- very dark productivity UI
- heavy national-style decoration
- inconsistent card sizes

## Component Priorities

- sidebar should stay stable and readable
- search should be top-level and obvious
- primary actions should be visible but not noisy
- cards should feel ordered and easy to scan

## Copy Rules

- generic UI text in Chinese
- preserve English brand names for product/tool names
- keep labels short and practical

## Visual Consistency

Reuse the same:

- radius scale
- border softness
- glass background opacity
- shadow softness
- spacing rhythm
